Make your own Parabo's Game Quiz

 October 25, 2024

Parabo’s Game Quiz is a homebrew game for the Satellaview (the Super Famicom extension).
It is a quiz where you have to identify games from a blurry screenshot.

Here is the template you can use to make your own quiz.
Send me all the files and I will compile your own version of the game.

The original version has 12 rounds but you can make any number of rounds (within reason).

I’ll add you to the credit screen as quiz creator.

game-list.txt


This is a .txt files of all the game listed in the answer selection screen.
Format: 1 game per line
Restriction: 28 character-long titles In game, the list is paginated with 20 games on each page. The total number doesn’t have to be a multiple of 20. I just thought that 20 pages was not too cumbersome to navigate.

Example (400 NES games): game-list.txt


game-info.txt


This is a .txt file that contains the answers info.
Format: 1 game per line, semicolon separated fields
Fields (in order): round, title, year, genre, developer

Restriction:
Title: 28 characters long
Genre: one or two 16-characters long lines (use / to indicate the line break)
Developer: one or two 21-characters long lines (use / to indicate the line break) \

Example: game-info.txt


Images


Each game uses 3 images.
Resolution: 224*128.
The MC and NPCs images needs a transparent background (or a magenta fill with #FF00FF).
Restriction: No overlap between the MC and the NPCs.
