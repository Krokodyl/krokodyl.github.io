Treasure Conflix - All Treasure Locations

 July 10, 2023

There are 32 treasures to find in the game (not counting the Cloud Gate, which is the main story element).
20 of the treasures are found with a clue given by an NPC.
8 of the treasures are found using a treasure map given by an NPC.
4 of the treasures are underwater in the Doom Triangle.
