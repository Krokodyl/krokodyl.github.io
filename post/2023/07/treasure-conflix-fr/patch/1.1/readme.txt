French translation for the Satellaview (SNES/Super Famicom) game
Treasure Conflix released in 1996 by Squaresoft.

Version: 1.1

Known bugs:
- Sometimes Melk's sprite a bit is messed up

It applies to the following ROM :
File: BS Treasure Conflix
No-Intro: Super Nintendo Entertainment System (v. 20180813-062835)
ROM SHA-1: DFF1D22D662FED96F45AC3E722C79C93469CA421
ROM CRC32: F50FB0B7

------------------- THE GAME -------------------------------
The game mixes RPG elements with some airship fights in mode 7.
Most of the focus is on talking with NPCs and solving riddles
trying to find the locations of treasures.
